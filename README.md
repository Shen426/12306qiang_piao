# 🚅 12306 极速抢票助手 (Playwright 自动化版)

> **假期出去玩的票总是开售秒没怎么办？别怕，我来了！**

本项目是一个基于 Python 和 [Playwright](https://playwright.dev/python/) 框架构建的 12306 端到端自动化购票脚本。专为应对“五一”、“十一”及春运等极度拥堵的高并发节点设计。

**🤖 核心开发声明：** 本项目的核心架构设计、复杂 DOM 解析算法、反爬虫防风控策略的重构以及相关代码的编写，均是在 **AI 软件的深度辅助与协同下完成的**。

## ✨ 核心特性

- **🚀 零延迟空降查询**：利用 URL 参数动态拼接，直接绕过前端首页容易卡死的车站电报码输入框 Bug，直达查票页。
- **🛡️ 强力反风控与隐匿**：引入微秒级随机抖动延迟（Jitter Delay）并擦除 WebDriver 自动化特征，极大降低被 12306 防火墙拦截及弹出滑块的概率。
- **🧠 智能动态 DOM 溯源**：摒弃脆弱的绝对路径选择器，采用“文本穿透 + XPath 溯源”双重算法，精准且幂等地自动勾选乘车人。
- **🔄 突破熔断自愈机制**：在提交订单遭遇“排队人数超过余票”的官方熔断提示时，脚本能瞬间销毁无效标签页并无缝重启下一轮猛攻，不死闭环直到成功抵达支付网关。

## 🛠️ 安装指南

**1. 克隆仓库到本地：**

```bash
git clone https://github.com/Shen426/12306qiang_piao.git
cd 12306qiang_piao
```

**2. 安装 Python 依赖库：**
需确保已安装 Python 3.8 或以上版本。

```bash
pip install playwright
```

**3. 初始化 Playwright 浏览器内核：**

```bash
playwright install chromium
```

## 🚀 使用教程

1. **修改个人配置：**
   打开代码中的 `TICKET_AUTOMATION_CONFIG` 字典（或 `config.py`），填入你真实的抢票信息：
   - `travel_date`: 乘车日期 (格式如 `"2026-05-01"`)
   - `departure_station` / `departure_code`: 出发站及电报码 (如 `"南京"` 和 `"NJH"`)
   - `arrival_station` / `arrival_code`: 到达站及电报码 (如 `"徐州"` 和 `"XCH"`)
   - `target_trains`: 期望抢购的车次列表，按意向优先级排序 (如 `["G1252", "G634", "G1970"]`)
   - `passengers`: 你的真实姓名列表 (须与 12306 账号内绑定的姓名完全一致)

2. **启动抢票：**

   ```bash
   python qiang_piao.py
   ```

3. **扫码待命与极速冲刺：**
   - 建议在开售前 **5分钟** 运行脚本。
   - 浏览器自动弹出后，使用手机 12306 App 完成**扫码登录**。
   - 页面自动跳转至【个人中心】后，**切回代码终端按下键盘回车键（Enter）待命**。
   - 脚本将瞬间空降查票页面，全自动完成高频刷新、秒锁车次、自动选人及提交订单。
   - 听到终端打印出 `🎉🎉🎉 SUCCESS` 时，请立刻在限制时间内掏出手机完成付款！
